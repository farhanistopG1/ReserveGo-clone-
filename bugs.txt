Bug 1 (Biggest)

waitlist_log() returns one dictionary:

{
    "Name": customer_name,
    "Phone Number": phone_number,
    ...
}

Then you do

for entry in log:

But log is not a list.

It is a dictionary.

Looping over a dictionary gives its keys.

So entry becomes

"Name"

then

"Phone Number"

then

"Party Size"

...

NOT the customer.

Bug 2

Because of Bug 1,

this immediately breaks:

entry["Name"]

Why?

Because

entry == "Name"

which means Python tries

"Name"["Name"]

which is impossible.

Bug 3

This line has invalid quotes.

logging.info(f"Adding wait number {waitlist_number} for {entry["Name"]}")

Python sees

"

inside another

"

and thinks the string ended.

Use single quotes inside:

entry['Name']

or vice versa.
